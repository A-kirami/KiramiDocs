## 🚀 弹射起步

有手就行，只需几分钟，即可开始你的 KiramiBot 之旅！

在终端中执行以下命令，它将自动安装 [Kirami CLI](https://github.com/A-kirami/KiramiCLI)。

import TabItem from '@theme/TabItem'
import Tabs from '@theme/Tabs'

<Tabs groupId="platform">
<TabItem value="windows" label="Windows" default>

```powershell
(Invoke-WebRequest -Uri https://install.kiramibot.dev -UseBasicParsing).Content | py -
```

</TabItem>
<TabItem value="linux" label="Linux">

```bash
curl -sSL https://install.kiramibot.dev | python3 -
```

</TabItem>
</Tabs>

接下来运行以下命令创建你的 KiramiBot 项目：

```bash
kirami create
```

跟随提示完成配置，恭喜你，你已经成功创建了一个 KiramiBot 项目！

## ✨ 闪光点

- 🎁 **开箱即用**

  无门槛一键安装，无需复杂配置，小白也能轻松上手。

- 🎛️ **插件管理**

  非侵入式插件管理系统，支持精细的功能和权限控制，更易于维护和管理。

- 🤝 **开发友好**

  具备完整类型注解，包含常用功能模块，助力开发者专注于核心功能。

- 🍀 **生态兼容**

  原生支持 **[NoneBot2](https://v2.nonebot.dev/)** 插件，无需修改代码即可平滑迁移项目。

## 🪧 设计理念

- 🐣 **用户为本**

  KiramiBot 以简单易用为目标，即使没有编程经验的用户也能快速上手，零门槛体验。

- 💖 **开发友好**

  KiramiBot 旨在降低开发者的心智负担和学习成本，减少开发的复杂度和代码量，让开发更加轻松高效。

- ✍ **规范标准**

  KiramiBot 提供开发规范和统一标准，降低开发者的选择成本与维护成本。

- 🔋 **内置电池**

  KiramiBot 内置丰富功能，避免重复造轮子，大多数功能可通过导入内置模块实现。
