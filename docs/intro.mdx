## 🚀 弹射起步

有手就行，3分钟搭建好属于你的 KiramiBot！

在终端中执行以下命令，它会为你自动安装好 [Kirami CLI](https://github.com/A-kirami/KiramiCLI)。

import CodeBlock from '@theme/CodeBlock'
import TabItem from '@theme/TabItem'
import Tabs from '@theme/Tabs'

<Tabs>
  <TabItem value="windows" label="Windows" default>
    <CodeBlock language="powershell">
      (Invoke-WebRequest -Uri https://install.kiramibot.dev -UseBasicParsing).Content | py -
    </CodeBlock>
  </TabItem>
  <TabItem value="linux" label="Linux">
    <CodeBlock language="bash">curl -sSL https://install.kiramibot.dev | python3 -</CodeBlock>
  </TabItem>
</Tabs>

接下来运行下面的命令创建 KiramiBot 项目：

```bash
kirami create
```

跟随提示完成配置，恭喜你，你已经成功创建了一个 KiramiBot 项目！

## ✨ 闪光点

- 🎁 **开箱即用**

  - 无门槛一键安装，初次接触更易上手，小白也能即装即用。

- 🎛️ **插件管理**

  - 非侵入式的插件管理系统，支持更细粒度的功能与权限控制，更加便于维护和管理。

- 🤝 **开发友好**

  - KiramiBot 拥有完整的类型注解，同时提供了常用的功能模块，帮助开发者专注于功能本身。

- 🍀 **生态兼容**
  - 兼容 NoneBot2 原生插件，无需修改代码即可使用，无缝迁移项目。

## 🪧 设计理念

- 🐣 **面向用户** - KiramiBot 应简单易用，即使是毫无编程基础的用户，也可以快速上手，做到真正的零门槛。
- 💖 **开发友好** - KiramiBot 应降低开发者的心智负担与学习成本，同时减少开发时的复杂度与代码量，让开发更加轻松省力。
- ✍ **规范标准** - KiramiBot 应提供开发规范，统一标准，降低开发者的选择成本与维护成本。
- 🔋 **内置电池** - KiramiBot 应提供丰富的内置功能，让开发者不必重复造轮子，大多数需要的功能都能通过导入内置模块实现。
